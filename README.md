# TaskFlow - 定时任务调度管理系统

一个现代化的定时任务调度管理平台，支持 HTTP 请求、Shell 命令、Python 脚本和数据备份等多种任务类型。

## ✨ 功能特性

- **多类型任务支持**
  - HTTP 请求 (GET/POST/PUT/DELETE)
  - Shell 命令执行
  - Python 脚本运行
  - 文件备份 (支持压缩和自动清理)

- **智能调度系统**
  - 基于 Cron 表达式的灵活调度
  - 任务状态实时监控
  - 并发控制和任务队列
  - 失败重试机制

- **完善的日志系统**
  - 执行历史记录
  - 输出和错误捕获
  - 性能统计 (执行时长、成功率)

- **用户友好的界面**
  - 现代化 Web 界面
  - 实时数据更新
  - 响应式设计
  - 任务执行可视化
